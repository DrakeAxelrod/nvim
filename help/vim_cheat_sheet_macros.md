VIM CHEAT SHEET

Macros

| Key | Action               |
| --- | -------------------- |
| --- | -------------------- |
| qa  | record macro a       |
| --- | -------------------- |
| q   | stop recording macro |
| --- | -------------------- |
| @a  | run macro a          |
| --- | -------------------- |
| @@  | rerun last run macro |
